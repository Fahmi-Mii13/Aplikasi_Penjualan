# Aplikasi_Penjualan
Aplikasi Penjualan Toko ini merupakan sistem sederhana berbasis PHP Native dan MySQL yang digunakan untuk mengelola transaksi penjualan, data barang, dan manajemen user. Sistem ini dirancang untuk membantu proses bisnis toko agar lebih efisien, akurasi tinggi, dan mengurangi penggunaan pencatatan manual.
